 Lap charts

        
The horizontal axis is usually time or sample index, depending on the file. Speed is vehicle speed (typically km/h). Throttle / brake show pedal use (percent or estimated). Gear is the engaged gear. Together they show where you accelerate, brake, and how pace changes around the lap. Click a segment on the schematic map below to zoom the charts to that part of the lap.

        

          

          

          

        

      


      

        
 Track & segments

        
GPS plot: when the file contains plausible latitude/longitude, the scatter chart shows the path on the map plane. MoTeC fields named G_LAT / G_LON are often not GPS (they can be lateral/longitudinal acceleration channels), so they are ignored for mapping. Distance fallback: if there is no usable GPS but there is a Distance channel (metres travelled), the wide chart shows speed versus distance along the lap — not geography, but useful for pace vs position. Schematic segment map: the SVG is a fixed outline (Monza-style) for placing numbered segments. It is not your real GPS trace. Numbers line up with equal parts of the lap (by distance when available, otherwise by sample index). After you load segment deltas from two sessions, marker colours show where the compare lap is slower (orange) or faster (green) than the reference in each segment; grey means roughly matched. Values are approximate segment deltas in seconds (positive = you lose time vs reference in that segment).

 Lap comparison

        
Compare the same lap id (default lap-1) between two uploaded sessions for a quick summary. Use Load segment deltas on map after choosing reference and compare sessions to colour the schematic markers and show per-segment deltas. The segment model uses average speeds in each slice; it is a coaching-style hint, not a professional timing loop.
